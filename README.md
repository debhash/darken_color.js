darken_color.js
===============

Provide methods to parse color names/values and lightens or darkens it by passed amount value.

## Usage examples ##

    darken('lightgreen', '0.1'); // -> #63e763 
    darken('lightgreen', '10'); // -> #63e763
    darken('#90EE90', '10'); // -> #63e763
    darken('#9e9', '10%'); // ->  #98ed98
